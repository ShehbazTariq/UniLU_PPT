# Presentation rules

- Footer citations must use author/year text plus the paper title.
- Link each footer citation to the DOI or arXiv DOI URL when available.
- Cite a repeated paper only once at the first source-defining slide in a section or figure sequence. Do not add short repeated footers such as "Author et al., Venue Year" on follow-up explanation or result slides from the same paper.
- Do not show numbered reference labels in slide footers.
- Put multiple references on separate footer lines using `\framecite[2]{...}`, `\framecite[3]{...}`, and so on.
- Keep footer citations in the bottom-centre band between the footer logos. If content gets close to that band, move the content up or split the slide rather than raising the citation rows into the slide body.
- Use `compactblock` instead of plain `block` for stacked callouts or callouts in columns `<=0.45\textwidth`.
- Do not show local library IDs, private-note labels, source file paths, or implementation filenames as references in slide footers.
- Use literature terminology for QEC and other technical concepts. Avoid wording that can be confused with another established term unless the slide defines it.
- Use neutral provenance wording in action titles. Prefer reference baseline, reported curve, source data, or literature baseline.
